class arr2[A] extends Serializable

Utility class for a 2D squared array of same type objects

Author:

Stephane Plaszczynski

Example: 

  1. // some object:
case class Point(x:Double,y:Double,z:Double)
p1=Point(1,2,3)
// here is the arr2:
val a=arr2[Point](5)
// enter the matrix (note: do not need double parenthesis)
a(0,0)=p1
// access the element
print(a(0,0))
